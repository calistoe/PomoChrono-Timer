# 🍅 PomoChrono

PomoChrono is a productivity application inspired by the Pomodoro Technique, designed to help users stay focused, manage their time efficiently, and build consistent study or work habits. It combines a clean, modern interface with a simple workflow that encourages deep focus while preventing burnout through scheduled breaks.

Whether you're studying, working, or tackling personal projects, PomoChrono helps you stay on track—one session at a time.

---

## ✨ Features

- ⏱️ Customizable Pomodoro timer
- 🍅 Focus and break sessions
- ▶️ Start, pause, and reset controls
- 🎨 Clean and modern user interface
- ⚡ Lightweight and responsive desktop application
- 💻 Built with web technologies using ElectronJS

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- ElectronJS

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/PomoChrono.git
```

Navigate to the project folder:

```bash
cd PomoChrono
```

Install the dependencies:

```bash
npm install
```

If Electron is not installed:

```bash
npm install --save-dev electron
```

---

## ▶️ Running the Application

Start the application by running:

```bash
npm start
```

---

## 📖 How to Use

1. Launch the application.
2. Set your preferred focus duration (optional).
3. Click **Start** to begin your Pomodoro session.
4. Stay focused until the timer finishes.
5. Take a short break when prompted.
6. Repeat the cycle to maintain productivity.

---

## 🎯 Why PomoChrono?

The Pomodoro Technique is a proven time management method that improves concentration by breaking work into focused intervals followed by short breaks. PomoChrono simplifies this process through an intuitive desktop application, helping users develop productive routines while reducing distractions and mental fatigue.
